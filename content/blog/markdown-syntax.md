# Markdown Syntax Guide

This article offers a sample of basic Markdown syntax that can be used in Hugo content files.

## Basic Syntax

### Headings

```markdown
# H1
## H2
### H3
```

### Bold and Italic

```markdown
**bold text**
*italicized text*
```

### Lists

```markdown
1. First item
2. Second item
3. Third item

- First item
- Second item
- Third item
```

### Code

```markdown
`code`
```

````markdown
```javascript
console.log('Hello World!');
```
````

### Links and Images

```markdown
[Link](https://www.example.com)
![Image](image.jpg)
```

This guide covers the most commonly used Markdown syntax elements.
